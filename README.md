# House Price Prediction Project

## Описание проекта

Данный проект представляет собой веб-приложение для прогнозирования стоимости жилья на основе введённых пользователем данных. Прогноз осуществляется с использованием модели машинного обучения.

## Цель проекта

Целью проекта является разработка простой системы, позволяющей оценить стоимость недвижимости на основе ключевых параметров, таких как доход населения, возраст дома и другие характеристики.

## Используемые технологии

* Python
* Flask
* Pandas
* Scikit-learn
* Matplotlib
* HTML / CSS

## Структура проекта

* `app.py` — основной файл приложения
* `train_model.py` — обучение модели
* `predict.py` — логика прогнозирования
* `Housing.csv` — исходные данные
* `model.pkl` — сохранённая модель
* `templates/` — HTML-шаблоны
* `static/` — статические файлы (графики)

## Использование

Пользователь вводит следующие параметры:

* Средний доход района (Avg. Area Income)
* Средний возраст домов (Avg. Area House Age)
* Среднее количество комнат (Avg. Area Number of Rooms)
* Среднее количество спален (Avg. Area Number of Bedrooms)
* Население района (Area Population)

После ввода данных система рассчитывает и отображает прогнозируемую стоимость жилья.
